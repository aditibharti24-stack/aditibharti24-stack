## 📌 About Me

Hi, I'm **Aditi Bharti** — a B.Tech Computer Science student with a research-oriented mindset and a passion for building things that work *close to the metal* and *close to human perception*.

My work sits at the boundary of **low-level systems**, **artificial intelligence**, and **visual computing** — writing efficient code that enables machines to understand the world around them.

- 🔬 Researching — AI/ML, Computer Vision, and image processing pipelines
- 🛠️ Building — systems-level and full-stack applications
- 🤝 Collaborating — on AI/ML projects, web apps, and open-source
- 🌱 Currently exploring — edge AI, embedded CV, and kernel internals

## 🧠 Research & Technical Interests

### 🖥️ Systems & Low-Level Engineering
- Systems Programming in **C / C++** — memory models, concurrency, performance tuning
- Linux internals, OS architecture, and kernel-space development
- Embedded systems and hardware-software co-design

### 🤖 Artificial Intelligence & Computer Vision
- Deep learning for visual perception — detection, segmentation, image understanding
- Computational Photography and image processing pipelines
- Deploying CV models on constrained and real-time systems

### 🌐 Web & Full-Stack Development
- Backend API design and database architecture
- Full-stack applications as research interfaces and demos
- End-to-end pipelines from data to deployment
